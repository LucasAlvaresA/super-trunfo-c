# Super Trunfo em C

Programa desenvolvido em linguagem C que simula o cadastro e a comparação de duas cartas do jogo Super Trunfo, utilizando dados de cidades.

## Dados informados em cada carta

- Estado
- Código da carta
- Nome da cidade
- População
- Área
- PIB
- Pontos turísticos

## Cálculos realizados

O programa calcula automaticamente:

- Densidade populacional = população / área
- PIB per capita = pib / população
- Super poder = soma de todos os atributos + inverso da densidade

## Comparações feitas

As cartas são comparadas pelos seguintes critérios:

- População (maior vence)
- Área (maior vence)
- PIB (maior vence)
- Pontos turísticos (maior vence)
- Densidade populacional (menor vence)
- PIB per capita (maior vence)
- Super poder (maior vence)

## Como compilar e executar (Code::Blocks)

- Abra o Code::Blocks
-Clique em File → Open
- Selecione o arquivo do projeto ou o .c
- Clique em Build and Run (ou pressione F9)

O programa será compilado e executado automaticamente no console
